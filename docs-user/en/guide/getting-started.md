# Getting Started

## Logging in

Go to [https://recapp.uni-goettingen.de](https://recapp.uni-goettingen.de) and click **Login**. Sign in with your standard university credentials. You can set the interface language (German or English) on the login page before signing in.

No separate Recapp account is needed.

## The dashboard

After logging in, you land on the **dashboard** — your home page in Recapp. It shows all quizzes you're connected to: ones you've created, ones you've joined as a student, and ones where you've been added as a teacher. Quizzes are sorted by the date of the last change.

Each quiz card shows:
- Title and description
- Number of participants and questions
- Which teachers are assigned
- Current quiz status: **Editing**, **Started**, or **Stopped**

From the dashboard you can create a new quiz, import a quiz from a file, and quickly access, start, edit, share, or delete any existing quiz.

## Account settings

Your account settings are always accessible in the top-right corner. Click the circle showing your name initials to open the settings panel.

| Setting | What it does |
|---|---|
| **Pseudonym** | A display name to use instead of your real name when participating in quizzes. Click *edit* to set or change it. |
| **Participation options** | Your default name type when adding questions or comments: anonymous, pseudonym, or real name. |
| **Language** | Switch the interface between German and English at any time. |
| **Colour palette** | Choose between two colour schemes (red/green or blue/yellow) for accessibility purposes. |
| **Logout** | End your session. |

::: tip Setting a pseudonym
If you'd like to participate in quizzes anonymously or under a nickname, set your pseudonym before joining a quiz. It can be changed at any time.
:::
