# Fragen erstellen <Badge type="info" text="Studierende" />

Wenn das Quiz im **Bearbeitungsmodus** ist, können Sie Fragen einreichen, die Ihre Kommilitoninnen und Kommilitonen später beantworten werden. Das Formulieren von Fragen ist der Kern von Recapp — es fordert Sie auf, sich intensiv mit dem Lernstoff auseinanderzusetzen.

## Eine Frage hinzufügen

Klicken Sie im Tab **<mark>Quizfragen</mark>** auf **<mark>+ Neue Frage</mark>**.

**1. Namensanzeige wählen**  
Wählen Sie, wie Ihr Name bei der Frage erscheint: anonym, Pseudonym oder Klarname. Die verfügbaren Optionen hängen von den Einstellungen Ihrer Lehrperson ab. Die Lehrperson sieht immer, wer eine Frage eingereicht hat — andere Studierende nicht.

**2. Fragetyp auswählen**  
Nur von der Lehrperson aktivierte Typen stehen zur Verfügung:
- **<mark>Freitext</mark>** — Antwortende schreiben eine eigene Antwort; keine automatische Bewertung
- **<mark>Einfachauswahl</mark>** — eine richtige Antwort unter mehreren Optionen
- **<mark>Mehrfachauswahl</mark>** — eine oder mehrere Antworten können richtig sein

**3. Frage formulieren**  
Tippen Sie Ihre Frage in den Editor. Der Editor unterstützt einfaches Markdown — nutzen Sie die Symbolleistenschaltflächen oder klicken Sie auf **?** für eine Syntaxreferenz.

**4. Antwortoptionen hinzufügen** (bei Auswahlaufgaben)  
Klicken Sie auf **<mark>Antwort hinzufügen</mark>** für jede Option. Setzen Sie einen Haken bei den richtigen Antworten. Nutzen Sie die Pfeile zur Neuanordnung und das Minus-Symbol zum Entfernen einer Option.

**5. Hinweistext hinzufügen** (optional)  
Nutzen Sie das Hinweisfeld, um den Antwortenden einen Tipp zur Interpretation der Frage zu geben.

**6. Speichern**  
Klicken Sie auf **<mark>Frage speichern</mark>**. Ihre Frage erscheint in der Liste.

## Ihre Frage bearbeiten

Sie können Ihre eigenen Fragen bearbeiten, solange das Quiz im Bearbeitungsmodus ist. Klicken Sie auf das **Stift-Symbol** auf Ihrer Fragenkarte, nehmen Sie Änderungen vor und klicken Sie auf **<mark>Frage speichern</mark>**.

Sobald Ihre Lehrperson in den Kurationsmodus wechselt, ist eine Bearbeitung nicht mehr möglich.

## Was als nächstes passiert

Nachdem Sie Fragen eingereicht haben, prüft die Lehrperson diese. Sie kann Fragen bearbeiten, ausblenden oder neu anordnen, bevor das Quiz startet. Sie sehen Ihre eigenen Fragen in der Liste — aber nicht die Fragen anderer Studierender. Diese begegnen Ihnen erst, wenn das Quiz startet.

## Kommentare

Wenn Ihre Lehrperson Kommentare aktiviert hat, können Sie während der Antwortphase einen Kommentar zur aktuell angezeigten Frage hinterlassen. Weitere Informationen unter [Quiz beantworten](./answer-quiz#kommentare).
