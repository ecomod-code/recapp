# An einem Quiz teilnehmen <Badge type="info" text="Studierende" />

## So treten Sie bei

Ihre Lehrperson teilt einen **Link** oder einen **QR-Code** mit Ihnen. Klicken Sie auf den Link oder scannen Sie den QR-Code, um das Quiz zu öffnen.

Sie haben zwei Möglichkeiten:

**Mit Universitätskonto** — Klicken Sie auf **<mark>Anmelden</mark>** und melden Sie sich mit Ihren regulären Universitätsdaten an. Das Quiz erscheint in Ihrem Dashboard und Sie können jederzeit und von jedem Gerät darauf zurückgreifen.

**Ohne Anmeldung** — Sie können ohne Universitätskonto teilnehmen. Recapp erstellt eine temporäre Sitzung für Sie. Beachten Sie, dass diese an Ihren aktuellen Browser gebunden ist — wenn Sie den Tab schließen oder das Gerät wechseln, können Sie möglicherweise nicht dort weitermachen, wo Sie aufgehört haben.

## Was Sie beim Beitreten erwartet

Was Sie sehen, hängt vom aktuellen Quizstatus ab:

- <Badge type="warning" text="EDITING" /> — Wenn die Lehrperson es erlaubt hat, können Sie Fragen einreichen. Siehe [Fragen erstellen](./create-questions).
- <Badge type="tip" text="STARTED" /> — Sie gelangen direkt zur Antwortoberfläche. Siehe [Quiz beantworten](./answer-quiz).
- <Badge type="danger" text="STOPPED" /> — Das Quiz nimmt keine Antworten mehr entgegen. Sie können die Fragen und, wenn erlaubt, die Statistiken einsehen.

## Das Dashboard

Nach der Anmeldung zeigt das **Dashboard** alle Quizze, mit denen Sie verbunden sind. Jede Karte zeigt Titel, Anzahl der Teilnehmenden und Fragen, die Lehrperson(en) sowie den aktuellen Quizstatus.

Über das Dashboard können Sie jederzeit zu einem aktiven Quiz zurückkehren.

## Kontoeinstellungen

Klicken Sie auf den Kreis mit Ihren Namensinitialen oben rechts, um Ihre Kontoeinstellungen zu öffnen.

| Einstellung | Funktion |
|---|---|
| **Pseudonym** | Ein Anzeigename statt Ihres echten Namens. Legen Sie ihn fest, bevor Sie einem Quiz beitreten, wenn Sie anonym teilnehmen möchten. |
| **<mark>Teilnahmeoptionen</mark>** | Ihr bevorzugter Standardname: anonym, Pseudonym oder Klarname. |
| **Sprache** | Wechsel zwischen Deutsch und Englisch. |
| **<mark>Farbpalette</mark>** | Auswahl eines Farbschemas für Barrierefreiheit (Rot/Grün oder Blau/Gelb). |
